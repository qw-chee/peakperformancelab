# Peak Performance Lab

Peak Performance Lab is an interactive AI-powered learning application. It provides scenario-based activities to help students practise psychological skills such as growth mindset, self-talk, goal-setting, and mental imagery.

Live app: https://peakperformancelab.streamlit.app/

---

## Features

- Interactive, scenario-based learning modules  
- AI-generated feedback using OpenAI GPT-4o Mini
